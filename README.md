# 📊 Data Jobs Market Analysis

## 📖 Project Overview

This project analyzes data-related job postings to identify hiring trends, in-demand skills, top hiring companies, and preferred job locations. The analysis helps aspiring data professionals understand current industry requirements and make informed career decisions.

---

## 🎯 Objectives

- Analyze the most demanded technical skills.
- Compare skill requirements across different data roles.
- Identify top hiring companies.
- Analyze hiring locations.
- Categorize skills into domains.
- Compare common skills using a heatmap.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
